Full Narrative

Artifact Overview

The artifact selected for this enhancement is an Android-based Event-Tracking mobile application originally developed during the CS-360 Mobile Architecture and Programming course. The application allows users to create accounts, log in securely, and manage personal events using SQLite with full CRUD functionality and SMS reminders.

Justification for Inclusion

This artifact demonstrates software engineering skills including modular architecture, validation logic, database integration, and permission handling. The enhancements improved maintainability, usability, and application stability while aligning the artifact with industry mobile development practices.

Description of Enhancements

Enhancements focused on improving architectural separation, strengthening validation, refining UI behavior, and ensuring proper permission handling. These changes improved reliability and user experience while supporting scalable development practices.

Reflection

Enhancing this artifact reinforced the importance of defensive programming and maintainable design. Debugging permission handling and UI state transitions strengthened my understanding of Android lifecycle behavior and iterative testing practices.

  • Algorithmic Thinking: Structured CRUD workflows and UI state logic.
